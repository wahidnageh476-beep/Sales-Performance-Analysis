# Sales Dashboard Project

## 1. Project Overview
This project analyzes company sales and customer behavior. The goal is to provide actionable insights through an interactive *Power BI dashboard*, helping decision-makers understand trends, top-performing products, and regional sales performance.

---

## 2. Tools Used
- *Power BI Desktop* – for building interactive dashboards and visualizations  
- *SQL* – for querying and preparing data (optional)  
- *Excel / CSV* – source data  

---

## 3. Data
The dataset includes:
- sales_data.csv – Raw sales data including columns like Date, Product, Quantity, Price, Region, Customer ID

SQL queries used for data extraction and cleaning are included in sql/queries.sql.

---

## 4. Features
- *Key Performance Indicators (KPIs):*
  - Total Sales  
  - Number of Customers  
  - Average Sales per Customer  
  - Top Selling Products  
  - Monthly / Quarterly Sales Trends
- *Visualizations:*
  - Bar/Column charts for product and regional comparison  
  - Line charts for sales trends  
  - Cards for KPIs  
  - Slicers for filtering by product, region, and date

---

## 5. How to Use
1. Open dashboard/sales_dashboard.pbix in Power BI Desktop.  
2. Connect to the sales_data.csv file if prompted.  
3. Explore the dashboard using filters and slicers.
